# 堀内塗装 施工実績写真選定・分類シート

## 概要
このドキュメントでは、ホームページ用施工実績写真の選定基準と分類を管理します。
対象ファイル数: 128枚のJPG画像 (`website/images/converted_jpg/`)

---

## 分類カテゴリー

### 1. 外壁塗装（ビフォー・アフター）
外壁の塗装前後が分かる写真。色の変化、汚れの除去、艶の復活などが確認できるもの。

**優先採用基準:**
- 施工前・施工後の対比が明確
- 色の変化（白→新色、褪色→鮮やか など）
- 建物全体が映る（周図の状況が分かる）
- 高画質・適切な露出

### 2. 屋根塗装
屋根の塗装前後。瓦・ガルバリウム・スレートなどの素材別。

**優先採用基準:**
- 屋根全体が映る
- コケ・汚れの除去前後
- 色の変化が明確

### 3. 防水工事
ベランダ・屋上・外壁の防水工事。ウレタン防水・シート防水等。

**優先採用基準:**
- 施工前後の対比
- 防水層の状態が分かる

### 4. その他（シーリング・付帯等）
シーリング補修、ベランダ手すり、雨樋など周辺部材の塗装。

---

## ファイル命名規則（採用時）

採用された写真は以下の命名規則でリネーム:

```
works_{カテゴリ}_{連番}.{拡張子}

例:
- works_exterior_01.jpg (外壁塗装)
- works_roof_01.jpg (屋根塗装)
- works_waterproof_01.jpg (防水工事)
- works_other_01.jpg (その他)
```

## 次のステップ

1. **写真内容の確認**: 各写真を開いて内容を確認
2. **カテゴリー分類**: 上記4カテゴリーに振り分け
3. **優先度設定**: 採用優先度をA（必須）・B（推奨）・C（候補）で分類
4. **リネーム・圧縮**: 採用写真をweb用に最適化（幅1200px程度、画質80%）
5. **HTML更新**: `index.html` の施工実績セクションに追加

---

## 注意事項

- **重複ファイル注意**: `IMG_XXXX (1).jpg` と `IMG_XXXX.jpg` は重複の可能性あり。画質の良い方を採用。
- **肖像権・プライバシー**: 人物が写り込んでいる場合は確認が必要
- **住所・車両ナンバー**: プライバシー情報が含まれる場合は加工または非採用
- **著作権**: すべての写真は堀内塗装の施工写真であることを確認
